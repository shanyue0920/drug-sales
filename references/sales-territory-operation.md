# 区域规划·指标运营·纯销渠道·SFE（维S2·S4·S7）

> 本 reference 解决"指标怎么拆、纯销怎么盯、数据怎么用"。口径须标注（终端价/纯销/计量单位）。

## S2 区域规划与指标运营

### S2.1 指标拆解逻辑
`区域指标 = Σ(医院基线纯销 × 增长系数)`，其中增长系数由：
- 产品生命周期（新上市高增、成熟低增、VBP后靠增量科室）
- 区域潜力（未满足需求/竞品份额）
- 历史达成与团队能力
综合得出。拆解到**代表→医院→季度→月**，保留口径说明。

### S2.2 区域漏斗
覆盖医院数 → 有效拜访数（达观念阶段推进） → 观念渗透（认同+） → 处方启动（首方） → 纯销转化（持续处方）。
- 漏损诊断：哪一层掉得最多就补哪层（覆盖低→加人/加频；渗透低→强学术；启动低→解准入/可及）。

### S2.3 资源分配
- 预算/学术活动按"医院潜力 × 观念阶段缺口 × 活动 ROI"分配，倾斜核心+高潜。
- 避免平均主义：空白医院先投准入（接 drug-access），成熟医院投增量科室。

## S4 纯销与商业渠道

### S4.1 流向读取与核对
- 商业流向（经销商出库/医院入库）→ 核对纯销（医院实际销售给患者）。
- 标口径：终端价 vs 出厂价、计量单位（盒/支/疗程）。

### S4.2 库存与断货
- 安全库存 = 月均纯销 × 提前期系数；效期预警（近效期先出）。
- 断货预警：商业库存 < 安全库存即触发补货协同。

### S4.3 纯销掉量归因
| 归因 | 信号 | 动作 |
|---|---|---|
| 断货 | 商业库存骤降 | 补货协同 |
| 竞品 | 竞品纯销升/份额降 | 强化差异化证据 |
| 观念 | 拜访渗透降 | 加学术活动 |
| 渠道 | 医院进货减 | 查准入/医保（接 drug-access） |

## S7 数据驱动与SFE

### S7.1 CRM 客户洞察
- 客户分级（按纯销贡献+潜力）→ 拜访有效性（有效拜访占比/观念推进率）。
- 预警：高潜低覆盖、高覆盖低处方、掉量客户。

### S7.2 流向透视
- 医院/科室/客户纯销趋势（同比/环比）→ 份额变化 → 竞品对标。

### S7.3 人效看板
- 人均产出、覆盖率、活动 ROI、指标达成率；标分母定义防歧义。
- 与 drug-hr 接口：人效模型归 drug-hr，本维出业务数据。
