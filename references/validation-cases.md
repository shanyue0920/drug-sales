## 案例一 · 卡度尼利单抗（开坦尼）· 新上市肿瘤创新药院内落地

### 1. 元数据头
- 分析对象：卡度尼利单抗注射液（开坦尼，PD-1/CTLA-4 双抗）
- 治疗领域/适应症：复发/转移性宫颈癌（2022-06-29 获批）；胃癌一线（2024-09）、宫颈癌一线（2025-05）陆续获批
- 分析视角：客户与终端管理（S1）+ 专业拜访/科室会（S3）+ POA（S5）
- 数据截止：2026-07；证据等级：L1（NMPA/医保局/企业官网）

### 2. 执行摘要
1. 🔍 卡度尼利 2022-06-29 获 NMPA 批准用于含铂化疗失败的复发/转移宫颈癌，为全球首个 PD-1/CTLA-4 双抗（来源：康方官网、NMPA）。
2. 🔍 复发/转移宫颈癌适应症于 2024 年纳入国家医保目录（2025-01 生效）；3 项适应症均入医保（来源：国家医保局 PPT、同花顺 2025-01）。
3. 🔍 目标人群约 3.8 万复发/转移宫颈癌患者/年，5 年生存率 <17%；AK104-201 研究 ORR 33%（来源：国家医保局信息摘要、林巧稚妇科肿瘤学）。
4. 💭 销售侧：团队/指标/纯销为「用户提供」，本示范以执行动作框架呈现，不填内部数字。

### 3. 主体（按 S1/S3/S5）
- **S1 客户与终端**：核心医院=省级肿瘤专科医院+三甲妇瘤科；KOL 地图分「处方决策者（妇瘤科主任）/影响者（CSCO 指南专家）/执行者（主治医师）」；进院后首科突破=妇瘤科。
- **S3 拜访执行**：六步法落到「双抗机制差异化（vs 单免）+ 医保后可及性」学术信息；科室会讲者须具资质、内容经 MLR 审核。
- **S5 POA**：月度 POA 以「指南一线推荐渗透 + 医保落地后新患启动」为主目标，学术会议高频低耗。
- **边界**：拜访技能训练内容归 drug-training T4；医学证据由 drug-medical-postlaunch 提供。

### 4. 关键结论与行动建议
- 行动：① 核心医院妇瘤科进院后 90 天首方突破计划；② 以 CSCO 指南首位推荐（复发转移二线免疫）做观念渗透；③ 医保落地窗口抢新患起始。

### 5. 不确定性 · 待核实清单
- ⚠️ 待核实：各适应症最新医保支付限定与双通道落地进度，须以最新地方医保平台复核。
- 💭 推测：区域潜力/指标拆解的「增长系数」须以企业历史基线+产品生命周期提供，禁止凭空给数。

### 6. 信息来源与证据等级
- 康方生物官网 开坦尼产品中心（获批/适应症）🔍 L1
- 国家医保局信息摘要 PPT（卡度尼利，YPSW202400179）🔍 L1
- 同花顺/企业新闻（2024 进医保）🔍 L2

## 案例二 · 司美格鲁肽（诺和泰/诺和盈）· 成长期产品院内处方拓展

### 1. 元数据头
- 分析对象：司美格鲁肽注射液（诺和泰 Ozempic·降糖；诺和盈 Wegovy·减重）
- 治疗领域：2 型糖尿病 / 长期体重管理（GLP-1 RA）
- 分析视角：区域规划与指标运营（S2）+ 纯销与渠道（S4）+ 合规红线（S6）
- 数据截止：2026-07；证据等级：L1（NMPA/国家医保局）

### 2. 执行摘要
1. 🔍 司美格鲁肽 2021-04 中国获批用于成人 2 型糖尿病，当年即通过谈判纳入国家医保目录（来源：国家医保局 2024-04-16）。
2. 🔍 医保支付范围限定「与成人 2 型糖尿病有关的两种情形」，减重（诺和盈 2024-06-25 获批）不在报销范围（来源：国家医保局、中国新闻网 2024-06-26）。
3. 🔍 2024 年 1-2 月全国医保报销超 70 万人次；口服制剂（诺和忻）2024-01 获批（来源：国家医保局、中山一院访谈）。
4. 💭 销售侧：区域指标/纯销为「用户提供」。

### 3. 主体（按 S2/S4/S6）
- **S2 指标运营**：降糖版（医保内）按「终端纯销」拆解；减重版（自费/未进医保）须区分口径，不得把减重需求计入医保报销指标。
- **S4 纯销渠道**：降糖版靠院内处方+双通道；减重版走自费，关注院外/DTP 与合规边界（禁诱导超适应症开方）。
- **S6 合规红线**：代表须严格 on-label 推广，不得暗示「减肥可刷医保」；学术活动讲者/证据合规。
- **边界**：医保支付规则深做归 drug-access；品牌策略归 drug-market。

### 4. 关键结论与行动建议
- 行动：① 指标拆解显式分离「降糖（医保）/减重（自费）」两口径；② 代表行为红线培训重点=医保支付范围 + 超适应症合规；③ 纯销诊断监控两版本流向差异。

### 5. 不确定性 · 待核实清单
- ⚠️ 待核实：诺和盈（减重）最新定价与各省挂网/双通道进度，须以最新官方平台复核。
- 💭 推测：区域降糖可及性受医保续约与产能影响，须以最新政策更新。

### 6. 信息来源与证据等级
- 国家医保局《"减肥神药"到底进没进医保？》（2024-04-16）🔍 L1
- 中国新闻网《减重版司美格鲁肽在中国获批》（2024-06-26）🔍 L1
- 中山一院/信息时报（两版本差异与报销）🔍 L2

## 案例三 · 硫酸氢氯吡格雷（信立泰·泰嘉）· VBP 出局后渠道转移

### 1. 元数据头
- 分析对象：硫酸氢氯吡格雷片（泰嘉，信立泰首仿）
- 治疗领域：抗血小板/心脑血管
- 分析视角：纯销与商业渠道（S4）+ POA（S5）+ 集采后应对（跨 P5 接口）
- 数据截止：2026-07；证据等级：L1/L2（联盟采购公告、权威媒体）

### 2. 执行摘要
1. 🔍 2019-09-24「4+7」扩围氯吡格雷 75mg 中标：石药欧意 2.44、赛诺菲 2.55、乐普 2.98 元/片；信立泰报 3.13 元/片出局（来源：赛柏蓝/健康界 2019-09-25）。
2. 🔍 信立泰为首轮 4+7 中标者，本轮降价不足致流标，公告 2.5-5 亿回购自救（来源：每日经济新闻 2019-10-07）。
3. 🔍 原研赛诺菲以近仿制药价中标，防御性保份额（来源：界面新闻）。
4. 💭 销售侧：出局后院外/零售放量须以企业年报/米内更新。

### 3. 主体（按 S4/S5）
- **S4 纯销诊断**：出局=院内纯销断崖；掉量归因=「集采未中标」非观念/断货；立即转商业流向监控至院外。
- **S5 POA**：重心由「院内处方」转「院外/DTP/零售 + 处方外流承接 + 创新药转型叙事」。
- **跨接口**：集采未中标应对归 drug-procurement P5；本维只出销售执行动作（渠道转移+活动重排）。
- **边界**：合规不输出规避集采方案；医学证据归 drug-medical-postlaunch。

### 4. 关键结论与行动建议
- 行动：① 院内团队转院外零售覆盖与 DTP 药店合作；② POA 改为「患者依从+长期管理」学术活动；③ 同步承接公司创新管线进院储备。

### 5. 不确定性 · 待核实清单
- ⚠️ 待核实：信立泰出局后各渠道实际纯销占比，须以企业年报/商业流向更新（非公开则标 💭）。
- 💭 推测：原研中标后院内份额变化须以最新米内份额复核。

### 6. 信息来源与证据等级
- 赛柏蓝/健康界《47 全国扩围 中标结果》（2019-09-25）🔍 L2
- 每日经济新闻/人民号《信立泰流标股价下跌》（2019-10-07）🔍 L2
- 界面新闻（赛诺菲降价动因）🔍 L2
