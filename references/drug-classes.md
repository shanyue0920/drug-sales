# 常用药理分类速查

> 本文件是 `本 skill 内置基础模块（foundation.md）` 的参考之一，供本 skill 在「机制与靶点」「药理分类」环节引用。
> 用途：快速定位药品所属大类、代表药与核心机制，避免机制描述错误。
> 注意：仅列常用骨架，具体药品靶点/机制须联网核实并标注 🔍/💭。

## 1. 降糖药

| 类别 | 代表药 | 核心机制 |
|---|---|---|
| 双胍类 | 二甲双胍 | 抑制肝糖输出、改善胰岛素敏感性 |
| 磺脲类 | 格列美脲 | 促胰岛素分泌（关闭 KATP 通道） |
| DPP-4 抑制剂 | 西格列汀 | 抑制 DPP-4，升高内源性 GLP-1 |
| GLP-1 受体激动剂 | 司美格鲁肽、利拉鲁肽 | 激活 GLP-1R，促胰岛素、抑胃排空、减重 |
| SGLT-2 抑制剂 | 达格列净、恩格列净 | 抑制肾糖重吸收，排糖利尿 |
| 胰岛素及类似物 | 门冬、甘精 | 补充/替代胰岛素 |
| 噻唑烷二酮类 | 吡格列酮 | 激活 PPAR-γ，增敏 |

## 2. 抗肿瘤药

| 类别 | 代表药 | 核心机制 |
|---|---|---|
| 细胞毒（烷化剂） | 顺铂、环磷酰胺 | DNA 交联，阻断复制 |
| 抗代谢 | 5-FU、卡培他滨 | 干扰核酸合成 |
| 微管抑制剂 | 紫杉醇、多西他赛 | 抑制微管解聚 |
| 拓扑异构酶抑制剂 | 伊立替康、依托泊苷 | 阻断 DNA 复制 |
| 小分子 TKI | 吉非替尼、奥希替尼 | 抑制酪氨酸激酶（EGFR 等） |
| 单抗（靶向） | 曲妥珠单抗、贝伐珠单抗 | 结合特异性抗原（HER2/VEGF 等） |
| 免疫检查点抑制剂 | 帕博利珠单抗、纳武利尤单抗 | 阻断 PD-1/PD-L1 |
| ADC | 德曲妥珠单抗（T-DXd） | 抗体偶联毒素，靶向递送 |

## 3. 降压药

| 类别 | 代表药 | 核心机制 |
|---|---|---|
| ACEI | 缬沙坦（ARB）/ 依那普利 | 抑制 RAAS，扩血管 |
| CCB（二氢吡啶） | 氨氯地平 | 阻断 L 型钙通道，扩外周动脉 |
| β 受体阻滞剂 | 美托洛尔 | 抑制交感，降心率心输出 |
| 利尿剂 | 氢氯噻嗪、吲达帕胺 | 排钠利尿，降容量 |
| ARNI | 沙库巴曲缬沙坦 | 脑啡肽酶抑制 + ARB |

## 10. 其他特药（骨架）

- **血液**：促红素（EPO）、铁剂、升白（G-CSF）；抗凝见第 5 节。
- **眼科**：抗 VEGF（雷珠单抗、阿柏西普）治黄斑变性；前列腺素类（拉坦前列素）降眼压。
- **罕见病 / 特药**：酶替代疗法、单抗特效药、部分基因疗法——具体分析时联网核实机制与 NMPA/FDA 获批状态。

> 用法提示：分析某药时，先在此表定位大类，再联网核实该药具体靶点、代谢酶（CYP 系）、主要不良反应，全部标 🔍/💭。
