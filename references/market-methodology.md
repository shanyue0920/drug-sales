# 市场测算方法论（速查）

> 本文件是 `本 skill 内置基础模块（foundation.md）` 的参考之一，供本 skill 在「患者流漏斗」「政策准入与医保集采」
> 「支付方与药物经济学」「SWOT与差异化定位」环节引用。
> 用途：统一市场分析的测算口径与模板，避免量级错误与凭空结论。

## 1. 患者流漏斗（Patient Flow Funnel）

把"潜在患者"到"规范治疗达标"逐层拆开，定位流失瓶颈：

```
总人口
  └─ 高危人群（危险因素暴露）
       └─ 患病人数（患病率）
            └─ 知晓率 → 就诊率
                 └─ 确诊率（受诊断能力影响）
                      └─ 治疗率（受可及性/支付影响）
                           └─ 依从率（受观念/副作用影响）
                                └─ 达标率（受方案/随访影响）
```

**各环节常见瓶颈与含义**：
- 低知晓/就诊率 → 疾病认知与筛查不足（如慢病、罕见病）。
- 低确诊率 → 基层诊断能力弱、缺少检测手段。
- 低治疗率 → 支付门槛、进院难、医生观念保守。
- 低依从/达标率 → 副作用、用药复杂、患者教育缺位。

**输出要求**：给出各环节大致比例（标 🔍 来源 / 💭 估算），并指出"最关键瓶颈环节"。

## 2. 市场规模测算

### 2.1 自下而上（Bottom-up，推荐用于单品/细分）
```
市场规模 = 适用患者数 × 渗透率(用药率) × 年均治疗费用
```
- 患者数：流行病学数据（患病率×人口）。
- 渗透率：当前/预期治疗率。
- 年费用：中标价/医保支付价 × 年用药量。

### 2.2 自上而下（Top-down，用于验证）
```
市场规模 = 总治疗领域市场 × 该产品预期份额
```

### 2.3 口径必须显式标注（防量级错误）
- **价格口径**：终端价（医院/零售）/ 出厂价 / 医保支付价。
- **地域口径**：中国 / 全球 / 特定区域。
- **人群口径**：确诊患者 / 估测总患病人群。
- **周期**：年化市场。
- 估算项统一标「估算」。

## 3. 药物经济学要点

- **核心指标**：ICER（增量成本效果比）、QALY（质量调整生命年）。
- **支付方关注**：预算影响、与标准疗法成本效果对比、是否在阈值内（常以人均 GDP 倍数衡量）。
- **常用证据**：模型研究（Markov 模型）、真实世界成本分析。
- 标注来源与模型假设，避免把厂商模型当独立结论。

## 4. SWOT 与差异化定位模板

### 4.1 SWOT 矩阵
| | 有利 | 不利 |
|---|---|---|
| **内部** | **S 优势**（临床数据、剂型、品牌） | **W 劣势**（适应症窄、价格高、进院难） |
| **外部** | **O 机会**（指南更新、医保扩容、未满足需求） | **T 威胁**（集采、竞品上市、替代疗法） |

### 4.2 差异化定位陈述（Positioning Statement）
```
针对 [目标人群/适应症]，
[产品名] 是 [类别]，
其核心差异在于 [关键临床/可及性优势]，
相比 [主要竞品] 能为 [医生/患者/支付方] 带来 [具体价值]。
```

**输出要求**：SWOT 每一项必须能回溯到前述「临床/市场/政策」事实，不得凭空得出；定位陈述须可落地到一句传播语。

### 4.3 真实范例（减肥/代谢赛道：GLP-1 周制剂新药进入中国市场）
> 以下为结构范例，具体数字须按 foundation.md §2 联网校准；本范例仅演示「事实 → 结论」的推导链。

| | 有利 | 不利 |
|---|---|---|
| **内部** | **S 优势**：周制剂依从性高；STEP 系列 RCT 体重平均降 15–17%（🔍）；心血管保护信号 | **W 劣势**：原研定价高；首次进中国需自建渠道与医生教育 |
| **外部** | **O 机会**：中国超重/肥胖人群基数大、知晓治疗率低（患者流前段漏损大 = 增量空间）；体重管理需求上升 | **T 威胁**：专利到期临近、本土仿制密集排队；竞品双靶点（替尔泊肽）疗效更优；医保对「lifestyle 减重」覆盖有限 |

**差异化定位陈述**：
```
针对 [中国超重/肥胖且合并代谢风险、需长期体重管理者]，
[本 GLP-1 周制剂] 是 [每周一次给药的 GLP-1 受体激动剂]，
其核心差异在于 [显著减重幅度 + 心血管获益证据 + 周制剂依从性]，
相比 [日制剂竞品 / 尚待上市的本土仿制] 能为 [患者] 带来 [更优的长期体重管理与心血管保护]，
并为 [支付方] 在 [肥胖相关并发症] 上节省长期成本。
```

**可落地传播语（示例）**：「一周一次，科学减重，护心更长。」（须基于前述证据，不得夸大）

## 5. 关键市场机会与风险分析方法

**核心要求**：本节必须产出"能落到商业动作"的结论，禁止只写方向性口号（如「扩大适应症」「提升渗透率」「加强医生教育」）。每条机会须含「量级 + 杠杆点」，每条风险须含「落地抓手」。

### 5.1 机会池测算（自下而上，先有量级）
```
可服务患者池 = 目标人群规模 × 该人群中适用比例(估算) → 锚定机会量级
```
- 目标人群取已核实流行病学数据（如血透 102.7 万、CKD 患病率等，标 🔍）。
- 适用比例无一致数字时，用区间估算并标 💭（如「高钾在 MHD 中 20%–50%，可服务池约 20–59 万」）。
- 目的：让「机会多大」有锚点，而非空谈「市场广阔」。

### 5.2 机会的「反向逻辑」
- 低诊断率 / 低治疗率（如诊断 2.53%、治疗 8.69%）看似「市场小」，实则**反向说明渗透空间极大** → 这是存量蓝海，不是利空。
- 必须显式点出：「当前转化率低 = 未来增量空间」，避免把瓶颈只当风险。

### 5.3 每条机会须带「杠杆点 / 抓手」
机会不是结论，是「做什么能拿到」。每条机会后必须跟 1 个可执行商业杠杆，例如：
- 新适应症放量 → 杠杆：目标科室/中心准入 + 该科室医生教育。
- 支付红利 → 杠杆：双通道 / DTP 绕开单院药事会瓶颈。
- 刚需绑定 → 杠杆：心肾联合门诊 / MDT / 质控项目筛查。
- 反向蓝海 → 杠杆：联合质控中心把「筛查常态化」，把无症状人群变可管理。

### 5.4 风险须带「落地抓手」（禁止空话）
风险后必须跟应对动作，用表呈现：
| 风险 | 落地抓手 |
|---|---|
| 专利到期 / 仿制 / 集采 | 独占窗口内建「循证 + 指南 + RWE + 品牌心智」四件套；布局差异化剂型 / 复方延生命周期 |
| 系统瓶颈（低诊断治疗） | 绑定质控中心 / 登记网络做筛查教育与 KPI，纳入慢病管理路径 |
| 低价替代 | head-to-head RWE 说服支付方看「总治疗成本」而非单盒价 |
| 依从 / 安全性 | 患者教育 + 用药指导降低脱落 |

### 5.5 输出骨架（供模板 §4.3 使用）
- **机会池锚点**：可服务患者池量级（🔍/💭）+ 当前转化率 → 渗透空间。
- **TOP 机会**（每条：量级 + 杠杆）：编号列出，格式「机会描述 → 杠杆：具体商业动作」。
- **TOP 风险与应对**（表：风险 → 落地抓手）。

**反例（禁止）**：「机会：扩大适应症、提升渗透率、加强医生教育。」（无量级、无抓手，等于没说）

## 6. 常见陷阱

- 把"全球市场"当"中国市场" → 必须标地域口径。
- 患者流各环节比例拍脑袋 → 估算项标 💭 并说明假设。
- SWOT 写成了愿望清单 → 每条需有事实支撑。
- 用厂商经济模型直接下"划算"结论 → 注明模型来源与局限。
