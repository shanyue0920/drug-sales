# 循证证据等级标准（速查）

> 本文件是 `drug-base` 共享底座的 references 之一，供各角色 skill 在涉及「适应症与循证证据」「产品临床价值与证据」环节引用。
> 用途：统一证据等级的判定与标注，避免把低质量证据当高等级呈现。

## 1. 研究类型证据金字塔（由强到弱）

| 等级 | 研究类型 | 说明 |
|---|---|---|
| 1a | 系统评价 / Meta 分析（RCT） | 多个高质量 RCT 的合并分析，证据最强 |
| 1b | 单个随机对照试验（RCT） | 金标准干预性研究 |
| 2a | 队列研究（前瞻/回顾） | 观察性，可评估预后/风险 |
| 2b | 病例对照研究 | 回顾性，适合罕见结局 |
| 3 | 病例系列 / 横断面研究 | 无对照，描述性 |
| 4 | 专家意见 / 病例报告 / 动物实验 | 最低级别，仅作线索 |

**判定要点**：
- 干预效果（疗效/安全性）→ 优先看 RCT 与 Meta。
- 预后/流行病学关联 → 队列研究可接受。
- 单篇病例报告只能作为「信号」，不能作为结论。

## 2. 指南推荐级别

### GRADE 体系（国际通用）
- **强推荐 / 高质量证据（A）**
- **强推荐 / 低质量证据**
- **弱推荐 / 高质量证据**
- **弱推荐 / 低质量证据（D）**

### 中国指南常用写法
- **推荐类别**：I 类（已证实/一致认为有用）、II 类（有用性证据矛盾/分歧）、III 类（已证实无用/有害）
- **证据级别**：A（多中心 RCT/Meta）、B（单中心 RCT/大样本观察）、C（专家共识/小样本）

**判定要点**：标注指南时一并写出「推荐类别 + 证据级别 + 指南名称 + 年份」，例如「2024 ADA 指南，A 级推荐」。

## 3. 标注规则（与 drug-base §3 数据校准规则一致）

- 🔍 **已查证**：数据可追溯到官方说明书 / NMPA·FDA / 权威指南（如 ADA、ESC、NCCN）/ 核心文献（NEJM、Lancet、JAMA、NEJM 等）。
- 💭 **推测或不确定**：模型推断、单一非权威来源、信息冲突无法定论。

**示例**：
- 「司美格鲁肽减重适应症获 NMPA 批准（2024）🔍」
- 「该适应症国内年治疗费用约 1.2 万元（估算，按周制剂×52 周推算）💭」

## 4. 循证三件套罗列规范（强制）

**严肃医疗场景下，循证证据是报告可信度的底座。** 每一药品的「适应症与循证证据」与「产品临床价值与证据」都必须同时罗列以下三件套；任一缺失须标 💭「待核实」，**禁止用单源/厂商材料替代独立循证**。

### 4.1 研究数据（每条含字段）
每项研究须结构化呈现，不得只写「有 RCT 证实」：
- **关键 RCT**（核心疗效/安全证据）：【研究名(如 SUSTAIN/HARMONIZE) · 设计(随机双盲/开放) · 样本量(N=) · 关键终点结果(效应量/率) · 发表年份 · 证据等级(1a/1b)】
- **支持性研究 / Meta**：补充证据，同样字段化；Meta 须注明纳入研究数与异质性。
- **真实世界证据(RWE)**：若有，注明数据来源(登记库/回顾性队列)与样本量。
- 无头对头研究时，明确写「暂无头对头证据，以下为间接比较/分别证据」，不得暗示等效。

### 4.2 指南推荐（每条含字段）
- 【指南/共识名 · 发布机构(如 ADA/ESC/KDIGO/中华医学会) · 年份 · 推荐类别+证据级别(如 I/A、IIa/B) · 与本品相关的具体表述】
- 须写出该药在指南中的**具体地位**（如「推荐用于 XX 人群的一线/二线选择」），而非仅「被提及」。

### 4.3 专家共识（单独罗列）
- 指南常滞后或覆盖不全，本土化/实操建议多在专家共识中 → **必须单独列出**：
- 【共识名 · 发布机构(如 中华医学某分会/专家组) · 年份 · 与本品相关的核心建议】
- 例：「中国高钾血症管理专家共识(202X)，推荐新型钾结合剂用于长期控钾」。

### 4.4 字段缺失处理
- 任一字段查不到 → 标 💭 并写「待核实(来源/年份)」，**禁止编造研究名/样本量/NR**。
- 仅厂商材料、无独立文献/指南支撑 → 标 💭，不得当作 🔍 已查证。

## 5. 常见陷阱

- 把药企新闻稿当「已获批」→ 必须查 NMPA/FDA 官方数据库。
- 把单篇 II 期研究当「疗效确证」→ 标明阶段，等 III 期/获批。
- 将「显著」与「临床意义」混淆 → 注明效应量与置信区间。
- 循证只写「有 RCT 支持」一句话、不展开三件套字段 → 必须按 §4 结构化罗列。
